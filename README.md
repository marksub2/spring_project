API 명세
HTTP Method	API 경로 (Endpoint)	예상 역할 및 기능 설명
DELETE	/menu/delete/{idx}	
특정 인덱스(idx)를 식별자로 사용하여 해당 메뉴를 삭제합니다. 

GET	/menu/{idx}	
특정 인덱스(idx)에 해당하는 메뉴의 상세 정보를 조회합니다. 

GET	/menu/all	
등록된 모든 메뉴의 목록을 조회합니다. 

POST	/menu/add	
새로운 메뉴 정보를 시스템에 등록합니다. 

PUT	/menu/update/{idx}	
특정 인덱스(idx)의 메뉴 정보를 수정하거나 업데이트합니다. 

PUT	/menu/count/{idx}	
특정 메뉴의 수량이나 조회수 등 카운트 관련 데이터를 업데이트합니다.
